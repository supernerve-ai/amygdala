Why Amygdala?

Built for security teams drowning in alert noise.

⚡
Autonomous Triage

Feeds alerts into Foundation-Sec-1.1-8B for instant threat classification and severity scoring. Cuts alert noise by 90%+.

🔗
Event Correlation

Spawns investigation sub-agents that correlate events, trace lateral movement, and enrich context from surrounding logs.

📊
Structured Reports

Generates JSON incident reports and human-readable summaries pushed to Slack or any webhook in seconds.

🔒
Token-Based Auth

Connects to Splunk via MCP server using secure token-based authentication. TLS enforced in production.

🐳
Fully Containerized

Runs local or cloud with Docker. Zero vendor lock-in beyond Splunk. Deploy anywhere your infra lives.

🎯
Smart Escalation

No human needed unless the escalation threshold is hit. Configurable severity thresholds per alert category.

Quick Start

1
Clone
git clone https://github.com/supernerve-ai/amygdala.git && cd amygdala
2
Configure
cp .env.example .env.local

Fill in your Splunk credentials, MCP server URL, and model endpoint.

3
Install
pip install -e .

Or: pip install -r requirements.txt

4
Run
python -m amygdala.main

Or via Docker:

docker build -t amygdala . && docker run --env-file .env.local amygdala
